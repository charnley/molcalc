Virtual Molecular Model Kit

Introduction 

Welcome to the Virtual Molecular Model Kit (VMK). The VMK is designed to be used as a presentation tool and a resource for model kit experiments. The VMK has all the functionality of traditional teaching models and more. Models in 3D can be constructed using the Molecular Editor or the JME 2D to 3D Editor. The VMK connects to the NCI/NIH Chemical Resolver and PubChem. These services provide access to millions of unique chemical structures which can be loaded directly into the VMK display window. Models in the display window link to specific data sources via the AKA link of the Model Tools. All VMK control panels can be accessed from the Molecular Editor:
